Changelog

Release history for ShopGraph. Newest releases first.

April 13, 2026

v1.0.2
  • Free tier upgrade — Free tier now runs the full three-tier extraction pipeline (Schema.org + LLM + browser rendering). Limit changed from 500 Schema.org-only calls to 50 full-pipeline calls per month. Playground available at shopgraph.dev with 5 runs/day, no signup.

April 8, 2026

v1.0.1
  • AgentReady scoring — New 6-dimension scoring system (5 currently active, access_readiness stubbed at weight 0.00) for evaluating product data readiness for AI agents. Scores across six dimensions: completeness (structured_data_completeness), semantic richness (semantic_richness), UCP compatibility (ucp_compatibility), pricing clarity (pricing_clarity), inventory signals (inventory_signal_quality), and access readiness (access_readiness). Ships scoring methodology version 2026-04-08-v1.
    • Scoring methodology version: 2026-04-08-v1
  • Access readiness dimension — Stubbed at weight 0.00. Activates when Web Bot Auth adoption reaches detection threshold.
  • POST /api/score endpoint — Score existing product data without extraction.
  • include_score parameter — Add AgentReady scoring inline with any extraction request.
  • B2B scoring awareness — MOQ, lead time, and bulk pricing detection enhance scoring for B2B products.
  • Methodology documentation — Published versioned scoring methodology (2026-04-08-v1).

April 7, 2026

v1.0.0
  • Per-field confidence scoring — Every field in the response now includes a confidence score (0-1). Tier baselines: Schema.org 0.93, LLM 0.70, Hybrid 0.85.
  • strict_confidence_threshold parameter — Filter low-confidence fields from responses.
  • UCP output format — Add format=ucp to get Universal Commerce Protocol compatible output. Validated against ucp-schema v1.1.0.
  • Subscription billing — Starter ($99/mo, 10K calls) and Growth ($299/mo, 50K calls) tiers with API key authentication.
  • Auto-heal merge — Schema.org partial results automatically merged with LLM extraction for hybrid results.
  • Stripe MPP support — Agents can pay per-call via Stripe Machine-to-Machine Payments.
  • Field modifiers — Cross-validation, format checks, and provenance signals adjust per-field confidence.

March 24, 2026

v0.9.0
  • Initial launch — ShopGraph API and MCP server.
  • Schema.org extraction — JSON-LD and Microdata parsing for structured product data.
  • LLM extraction — GPT-powered extraction from raw HTML.
  • Playwright rendering — Headless browser for JS-heavy pages.
  • Playground — 50 calls/month, no signup required.
  • MCP server — Remote and local MCP server for Claude, Cursor, and other clients.
  • npm package — npm install shopgraph for programmatic access.
  • Test corpus — 208-URL test suite across 22 verticals (44% B2B / 56% B2C).
