FAQ

Frequently asked questions about ShopGraph.

Basics

ShopGraph is a product data extraction API that takes any product URL and returns structured, confidence-scored product data. It works with both B2C (e-commerce) and B2B (industrial supply) websites. Available as a REST API and MCP server.

ShopGraph extracts: product title, price, currency, brand, description, images, availability, SKU, GTIN/UPC, manufacturer part number, category, ratings, reviews, variants, and B2B fields (MOQ, lead time, bulk pricing). See the full ProductData schema.

Every field includes a confidence score from 0 to 1. The score starts with a baseline from the extraction method (Schema.org: 0.93, LLM: 0.70), then gets adjusted by field-level modifiers (cross-validation, format checks, etc.). You can set a strict_confidence_threshold to filter out low-confidence fields. See Confidence Scoring for details.

UCP (Universal Commerce Protocol) is a standard schema for commerce data interchange. When you add format=ucp to your request, ShopGraph returns data in UCP line_item format, validated against ucp-schema v1.1.0. See UCP Output.

AgentReady scoring evaluates extracted product data across 5 dimensions (completeness, confidence, structure, actionability, freshness) to produce a single score indicating how suitable the data is for autonomous agent workflows. See AgentReady Scoring.

Usage

The easiest way is to call the free API directly — no signup required. Send a POST request to https://shopgraph.dev/api/enrich/basic with a product URL. See the installation guide for all integration options.

ShopGraph works with any product page on the public web. It has been tested across 22 verticals covering both B2C (Shopify, Amazon, Best Buy, etc.) and B2B (Grainger, McMaster-Carr, MSC Industrial, etc.) with a 97% extraction success rate.

Pricing

Four tiers: Free (500/mo, $0), Starter (10K/mo, $99), Growth (50K/mo, $299), Enterprise (custom). Agents can also pay per-call via Stripe MPP without a subscription. See Pricing.

Stripe Machine-to-Machine Payments (MPP) lets AI agents pay per API call without a pre-purchased subscription. The agent passes a payment_method_id with each request and is charged $0.02 per full extraction.

Each POST to /api/enrich, /api/enrich/basic, or /api/enrich/html counts as one call. Cached responses (within TTL) do not count against your quota. Scoring-only requests (/api/score) are counted separately.

Enterprise

Enterprise plans include custom SLAs with guaranteed uptime, latency targets, and dedicated support. Contact hi@kb.computer for details.

The npm package can be used to run a local MCP server. Full self-hosting of the extraction API (including LLM and Playwright workers) is available on Enterprise plans.

ShopGraph caches extraction results for performance (configurable TTL). We do not store or sell extracted product data. API keys and usage metrics are stored for billing. See our Privacy Policy for full details.
