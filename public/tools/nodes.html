In the meantime, use the HTTP Request node in n8n to call the REST API directly:

n8n HTTP Request configuration
{
  "method": "POST",
  "url": "https://shopgraph.dev/api/enrich/basic",
  "headers": {
    "Content-Type": "application/json"
  },
  "body": {
    "url": "{{ $json.product_url }}"
  }
}
