MCP Server

Connect ShopGraph to any MCP-compatible AI client with one line of config.

Design Principle: Access without infrastructure. ShopGraph handles the extraction pipeline. Your agents get structured data with confidence scores. You don't manage proxies or identity stacks.

Configuration

Remote Server (recommended)

No installation required. Point your MCP client to the hosted endpoint:

MCP client config
{
  "mcpServers": {
    "shopgraph": {
      "type": "url",
      "url": "https://shopgraph.dev/mcp"
    }
  }
}

The remote MCP server works without an API key on the free tier (50 calls/month). For higher limits, add your API key as an environment variable and the server will include it automatically. See Pricing for tier details.

Local Server (self-hosted)

Run ShopGraph as a local MCP server via npx:

MCP client config
{
  "mcpServers": {
    "shopgraph": {
      "command": "npx",
      "args": ["-y", "shopgraph", "serve"],
      "env": {
        "SHOPGRAPH_API_KEY": "sg_live_..."
      }
    }
  }
}

Error Handling

MCP tool calls return errors as structured JSON objects with isError: true. Common error types: extraction_failed (page could not be parsed), payment_required (free tier exhausted, payment method needed), payment_failed (charge declined), ucp_mapping_failed (product data could not map to UCP format). Your agent can inspect the error field and retry or surface the message to the user.

Example: Agent Tool Call

A Claude or CrewAI agent calls ShopGraph as an MCP tool mid-workflow. The agent extracts product data, evaluates confidence, and decides whether to proceed or escalate to a human.

JSON
{
  "tool": "enrich_product",
  "arguments": {
    "url": "https://www.uline.com/Product/Detail/S-19318/Kraft-Paper/30-lb-Kraft-Paper-Sheets-8-1-2-x-11",
    "include_score": true
  }
}

One MCP tool call. The agent gets product data with confidence scores and an AgentReady score in a single response.

Payment

MCP calls on the free tier (50/month) require no authentication. For paid tiers, authenticate via:

  • API key — Set SHOPGRAPH_API_KEY in your MCP config env
  • Stripe MPP — Agents can pay per-call via Stripe Machine-to-Machine Payments
