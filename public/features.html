Features

ShopGraph is a product data extraction API that tells you when it is guessing. Every feature is built so agents can make trust decisions on the data they receive.

🎯

Confidence Scoring

Per-field confidence scores from 0 to 1. Tier baselines (Schema.org 0.93, LLM 0.70, Boosted 0.85) with field-level modifiers. Set a strict threshold to enforce data quality.
📦

UCP Output

Universal Commerce Protocol compatible output. Add format=ucp to get standardized line_item structures validated against ucp-schema v1.1.0.
🤖

AgentReady Scoring

5-dimension scoring for agent readiness: completeness, confidence, structure, actionability, and freshness. B2B-aware with MOQ and lead time detection.
🔌

Routing Engine

Intelligent extraction method selection. Schema.org, LLM, Playwright, and auto-heal merge. The system picks the cheapest method that meets quality requirements.
🛠

Self-Healing

Regression detection, circuit breaker patterns, confidence calibration pipeline, and LLM-as-validator. Tested against a 208-URL corpus (44% B2B / 56% B2C).
