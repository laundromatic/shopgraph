Where does ShopGraph fit?

🔗

From a URL

POST /api/enrich with any product URL. ShopGraph handles fetching, JS rendering, and extraction into a consistent schema. Every field includes a confidence score.

📄

From HTML

POST /api/enrich/html with raw HTML. Skip the fetch when you have your own scraping infrastructure. Same structured output, no redundant network calls.

🤖

From any agent

Connect via MCP with one line of config. Paid calls settle through Stripe MPP or API key subscriptions. Add format=ucp for Universal Commerce Protocol output.
