Integration

      
Works with Claude, Cursor, CrewAI, LangGraph, AutoGen, and any MCP client.

    


    

      

        

          
npm SDK:

          
import { ShopGraph } from 'shopgraph';
const sg = new ShopGraph();
const data = await sg.enrichBasic(url);

        

        

          
MCP Client:

          
{
  "mcpServers": {
    "shopgraph": {
      "type": "url",
      "url": "https://shopgraph.dev/mcp"
    }
  }
}

        

      

      

        REST API: POST /api/enrich/basic (free) · POST /api/enrich ($0.02) · POST /api/enrich/html ($0.02)
      

    


    

      
No API keys needed for the free tier. Paid calls use Stripe MPP. No subscriptions, no minimums.

    

  







  

    

      
Use Cases

    


    

      

        
Shopping Agents

        
An agent comparing products across the open web sends URLs or scraped HTML to ShopGraph and gets back structured data it can reason about.

      

      

        
Product Intelligence

        
An agent building recommendations or affiliate content gets structured attributes from source pages: price, availability, brand, category.

      

      

        
Catalog Ingestion

        
An agent maintaining a product catalog from multiple suppliers sends pages to ShopGraph for consistent structuring into one schema.
